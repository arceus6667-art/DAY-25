🚀 Tesla Stock Forecasting using ARIMA + Advanced EDA
📌 Overview

This project focuses on time-series forecasting of Tesla stock prices using ARIMA, combined with advanced exploratory data analysis (EDA) and interactive visualizations.

The goal is not just prediction — but understanding patterns, trends, and model limitations in real-world financial data.

📊 Features
📈 Time-series forecasting using ARIMA
🔍 Advanced EDA (histograms, scatter plots, correlation analysis)
🧊 3D visualization (Price vs Volume vs Time)
⚡ Interactive dashboards using Plotly
📉 Residual error analysis for model evaluation
📅 Date-based feature engineering
🧠 Key Learnings
Stock markets are highly volatile and noisy
ARIMA captures trend patterns, but struggles with sudden changes
Evaluation metrics like RMSE, MAE, MAPE are more reliable than accuracy
Visualization plays a crucial role in understanding model behavior
